# Pokémon TCG symbol assets

These local PNG files are used only to identify Pokémon TCG energy types and card rarities in the Card Detail interface. They were downloaded from the Bulbagarden Archives files embedded in the Bulbapedia [Type (TCG)](https://bulbapedia.bulbagarden.net/wiki/Type_(TCG)) and [Rarity](https://bulbapedia.bulbagarden.net/wiki/Rarity) reference pages on 18 July 2026.
